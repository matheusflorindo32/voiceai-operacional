# VoiceAI Operacional

Plataforma inteligente de transcrição, análise e narração científica aplicada à Segurança Pública e Saúde Ocupacional.

## Funcionalidades

- Speech-to-Text (Whisper)
- Análise com LLM (OpenAI)
- Narração humanizada (ElevenLabs)
- Estrutura modular profissional

## Aplicação

Projeto aplicado ao estudo de desconforto musculoesquelético relacionado ao uso de fardamento operacional.

## Como rodar

1. Criar arquivo `.env` com suas chaves.
2. Instalar dependências:

pip install -r requirements.txt

3. Rodar:

python main.py
